The Barbershop Appointment and Scheduling System is a modern web application designed to streamline appointment booking and schedule management for barbershops. The system allows customers to easily book appointments online while enabling administrators and barbers to efficiently manage schedules, services, and bookings.

The platform reduces manual scheduling, prevents double bookings, and improves overall customer experience.

**Features**  
  Customer

    Book appointments online
    
    Select preferred barber
    
    Choose available date and time
    
    Select service (haircut, shave, styling, etc.)
    
    View upcoming appointments

  Admin
  
    Manage barbers
    
    Manage services and pricing
    
    View all appointments
    
    Cancel or update bookings
    
    Monitor daily schedules
  
  Barber
  
    View assigned appointments
    
    Manage availability
    
    View daily or weekly schedule
  
  **Tech Stack**
  Frontend
  
    React – UI framework
    
    Tailwind CSS – Utility-first CSS framework
    
    shadcn/ui – Accessible and customizable UI components
  
  Backend
  
    Supabase – Backend-as-a-Service for authentication, APIs, and database access
  
  Database
  
    PostgreSQL – Relational database for storing appointments, users, and services
